## Configurations

<!-- configs -->

| Key                      | Description                                           | Type      | Default |
| ------------------------ | ----------------------------------------------------- | --------- | ------- |
| `status-bar-btn.enabled` | Enable or disable the status bar button.              | `boolean` | `true`  |
| `status-bar-btn.btns`    | An array of button configurations for the status bar. | `array`   | `[]`    |

<!-- configs -->
